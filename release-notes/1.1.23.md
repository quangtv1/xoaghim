## Xóa Vết Ghim PDF v1.1.23

### Tính năng mới
- **Render 10 trang một (Sliding Window)** - Chỉ giữ 10 trang trong RAM (trang hiện tại ±5), giảm 60% bộ nhớ cho PDF lớn
- **AI Detection Preload** - Tự động preload model YOLO DocLayNet khi bật bảo vệ text, loại bỏ độ trễ 3-5 giây
- **Background File Preload** - Pre-render file tiếp theo trong batch mode, giảm thời gian chờ giữa các file
- **Lazy Page Count Loading** - Load số trang bằng ThreadPoolExecutor (4 workers), UI không bị đơ khi mở thư mục lớn
- **Phím tắt [ ]** - Phím `[` quay về file trước, phím `]` chuyển file tiếp theo (batch mode)
- **Chữ thập tùy biến** - Con trỏ chữ thập dài gấp 3, màu hồng (Tùy chọn +) / xanh lá (Tùy chọn -)
- **Phím tắt Delete** - Xóa vùng chọn đang được chọn bằng phím Delete/Backspace
- **Phím tắt Ctrl+Delete** - Hiện popup "Xóa vùng chọn" (Mac: Ctrl+Backspace)
- **Phím tắt Esc** - Hủy chế độ vẽ Tùy biến đang chọn

### Sửa lỗi
- **Fix TextProtectionOptions crash** - Sửa lỗi crash khi bật bảo vệ text (dict → dataclass attributes)
- **Fix _load_from_cache invalid call** - Sửa lỗi crash khi load file
- **Fix Panel centering** - Sửa lỗi căn giữa panel với các kích thước trang khác nhau

### Cải tiến hiệu năng
| Metric | Trước | Sau | Cải thiện |
|--------|-------|-----|-----------|
| PDF 100 trang | ~500 MB | ~200 MB | Giảm 60% |
| PDF 1000 trang | OOM | ~400 MB | Khả thi |
| Mở thư mục lớn | UI đơ | Mượt | Lazy loading |
| Toggle bảo vệ text | Chờ 3-5s | Instant | AI preload |
